# MB Crunchy Commerce v2.0 (Sprint 1)

This package contains the Sprint 1 foundation.

Structure:
- backend/
- frontend/
- assets/

Before running:
1. Create a Google Apps Script project.
2. Upload all .gs files.
3. Deploy as Web App.
4. Replace YOUR_APPS_SCRIPT_WEB_APP_URL in frontend files.
5. Upload your logo, product images and UPI QR.
